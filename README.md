# AmmoCheck

Immersive™ user experience mod for [S.T.A.L.K.E.R. Anomaly](https://www.moddb.com/mods/stalker-anomaly)

  

## Requirements

  

- **Core mod**: no hard dependency (works on vanilla Anomaly 1.5.3).

- **Magazine features**: ~~Anomaly Magazines Redux~~ now requires [GAMMA MAGS RELOADED](https://discord.com/channels/912320241713958912/1322655858240262304) by AndTheHeroIs

- **Fast Reload**: requires [Dynamic Reload Speeds](https://github.com/Verdatim25/Dynamic-Reload-Speeds/) by Verdatim
- Optional support for:
	- Bwais FireMode UI mod
